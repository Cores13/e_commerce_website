# eCommerce Website
## Made using
NodeJS, React, MongoDB

# Backend 
NodeJS

# Backend Modules
* bcrypt 
* cloudinary 
* cookie-parser 
* concurrently 
* cors 
* dotenv 
* express 
* express-fileupload 
* jsonwebtoken 
* mongoose

# Frontend
React with TypeScript

# Frontend Modules
* react-router-dom 
* axios 
* react-paypal-express-checkout

## Start backend Server
`npm install`

`npm run dev`

## Start frontend Server
`yarn install`

`yarn start`
